Projects

Video Captioning based on Contextual and Temporal Feature-based Generative Adversarial Network

This project presents a hybrid GAN-based model for generating accurate, fluent, and context-aware captions for videos by learning both spatial and temporal features. The system integrates Contextual GANs, which utilize CNNs like ResNet to extract static visual features from individual frames with attention mechanisms, and Temporal GANs, which use LSTMs or 3D CNNs to capture motion and sequence flow across frames. The generator-discriminator structure promotes the creation of natural, human-like captions. Using the MSR-VTT dataset, the model was trained with adversarial and cross-entropy loss and evaluated using BLEU, METEOR, ROUGE-L, and CIDEr scores. With a BLEU score of 55 and METEOR of 30, it outperforms traditional captioning approaches. This system has practical applications in accessibility, content summarization, subtitling, and real-time video analysis.

Insights into Cardiac Wellness: Using CNNs for Predictive Heart Disease Analysis

This project focuses on developing a deep learning model using Convolutional Neural Networks (CNNs) to accurately predict heart disease based on patient medical data. The model architecture includes convolutional and pooling layers to extract relevant features, followed by fully connected layers for classification, with ReLU activation and dropout for regularization. Implemented in Google Colab, the model was optimized through hyperparameter tuning techniques like grid and random search. Despite challenges such as data noise and feature selection, the CNN demonstrated strong performance and generalizability. The system holds significant promise for early detection, personalized treatment, and remote monitoring, highlighting its potential for integration into smart healthcare and clinical decision-support systems.

Web Application for Ration Distribution Verification

Developed a web application to address the issues faced by users at ration shops when fingerprint machines fail. The application enables users to register with their details, including Aadhar card information and economic status. In case of a fingerprint machine malfunction, users can log in, generate a unique random number, and present it to the merchant for verification. The merchant verifies the random number through their portal, ensuring the user receives their rations without disruption.

Uber Data Analysis Using R

This project focuses on analyzing Uber ride data using R to uncover meaningful patterns in rider behavior, demand fluctuations, and geographic trends. The objective was to visualize and interpret ride timings across various time frames—hourly, daily, weekly, and monthly—and to forecast future demand. Data was cleaned and enriched using R libraries like dplyr, tidyr, lubridate, and visualized through ggplot2 and leaflet. Key analyses revealed peak usage during commute hours, weekend surges, and seasonal variations in demand. Geospatial heatmaps highlighted high-demand pickup zones, aiding in strategic driver placement. The study concluded with actionable insights to enhance Uber’s operational efficiency, support predictive modeling, and optimize rider-driver matchmaking for improved customer satisfaction.
